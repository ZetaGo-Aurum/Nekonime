Tentang NekoNime (ﾉ´ヮ´)ﾉ*:･ﾟ✧

Halooo Minna-san~ (≧∇≦)/ Selamat datang di NekoNime! Tempat paling uwu dan kawaii di jagat web buat kamu para pecinta anime sejati! ♡ඩ⌔ඩ♡

Dibuat dengan penuh cinta (´｡• ᵕ •｡`) ♡, NekoNime hadir biar kamu bisa cuci mata liat ilustrasi anime keren, moe, waifu/husbando-able dari berbagai sumber API pilihan (sekarang kategorinya makin banyak lho~!). Tampilannya modern next-gen pake glassmorphism di atas tema dark mode yang nyaman di mata, biar betah lama-lama di sini~

Fitur andalan kita yang bikin gemes:

  • Login & Cloud Save (Data aman di awan!)
  • Galeri Super Lengkap (SFW/NSFW, pilih API & kategori bejibun!)
  • Gacha Universal (acak dari semua kategori API, tanpa cooldown!⚡ Animasi baru!)
  • Tombol Muat Lebih Banyak Sepuasnya! (Anti-ngilang, klik terus!)
  • Bikin Koleksi Pribadi (harta karunmu!)
  • Simpan Favorit (buat yang spesial!)
  • Riwayat Jelajah & Gacha (kenangan manismu~)
  • Lightbox Detail Gambar (lebih dekat lebih mesra!)
  • Pilihan Sumber API (atur seleramu!)
  • Navbar Tetap di Atas (selalu setia menemani!)
  • Scroll Lancar Jaya (gak nyangkut lagi!)

Semua keajaiban ini berjalan mulus pakai Firebase, Tailwind CSS & JavaScript murni dalam Single Page Application (SPA). Cepat, ringan, dan uwu pastinya! (づ￣ ³￣)づ Jangan lupa kasih feedback ya kalau ada ide!
